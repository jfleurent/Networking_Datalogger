# Networking Datalogger

## Explanation for commands
Whenever running the server, instead of using `node index.js` use `nodemon index.js`. This will let the server keep running even when changes are being made.

### npm commands
* npm i init --yes
* npm i nodemon -g (optional)
* npm i express
* npm i mongodb

#### midleware
* npm i helmet --save
* npm i morgan --save
